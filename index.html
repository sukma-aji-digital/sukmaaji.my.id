Tentang Saya

Halo! Nama saya Aji, dan saya sangat suka membangun aplikasi web yang handal dan berskala. Minat saya pada teknologi dimulai sejak beberapa tahun yang lalu dan terus berkembang menjadi keahlian mendalam di bidang backend development, khususnya dengan PHP dan framework Laravel.

Tujuan saya adalah menciptakan aplikasi yang tidak hanya kuat dan efisien di balik layar, tetapi juga memberikan pengalaman yang mulus dan menyenangkan bagi pengguna akhir. Saya menikmati proses menjembatani logika server yang kompleks dengan antarmuka pengguna yang intuitif.

Berikut adalah beberapa teknologi inti yang saya kuasai:

  • MySQL & PostgreSQL
  • Desain REST API
